LLMの仕組み

— アニメーションで学ぶ、AIが言葉を生む原理 —

ChatGPTやClaudeのようなAIがどう動いているのかを、体験デモとともに一歩ずつ分解していく全6巻の教材シリーズです。①から順に進むのがおすすめですが、各巻は独立して読めます。

全6巻+講師用別紙 体験デモ 30種以上 HTMLだけで動作・通信不要
▼ 巻を選んで始める
①

推論編 — どうやって文章を書くのか

LLMの基本原理。「次の言葉あてゲーム」を軸に、トークン・確率・温度・自己回帰生成を体験します。
トークン / 埋め込み / softmax / サンプリング / 温度
②

学習編 — どうやって賢くなるのか

「次の言葉あて」の練習から始まる学習の全工程。損失・勾配降下・パラメータ・RLHFまでを、坂を転がるボールで理解します。
事前学習 / 損失 / 勾配降下法 / パラメータ / RLHF
③

Transformer深掘り編 — 中身を開けてみる

シリーズで最も踏み込んだ回。Q/K/Vのマッチング、マルチヘッド、MLP、層を重ねて意味が決まる過程を図解します。
Query/Key/Value / マルチヘッド / MLP / 位置エンコーディング
④

プロンプト編 — AIへの伝え方の科学

なぜ聞き方で答えが変わるのか。コンテキストウィンドウ、システムプロンプト、文脈内学習と、実践的な指示の型を学びます。
コンテキストウィンドウ / システムプロンプト / Few-shot / 指示の4要素
⑤

限界と対策編 — なぜ間違えるのか

ハルシネーションが起きる理由を仕組みから理解し、RAG・検証の判断軸まで。安全に使うための必修回です。
